Lists Elements

The types of lists include both ordered (ol) and unordered (ul) any listed items will be listed with the (li) element. The (ol) and (ul) elements both represent a list of items. The difference is with (ol) the order is meaningful.
Example:

  1. Steps for making your favorite cake!
  2. Directions to nearest Walmart to get ingredients!
  3. Rankings of your favorite Football Players!
You can change the (ol) where it changes the type of list is ordered like making the numbers become Roman Numeral!
Example using the type inside the (ol) can amke the list look like this:
  1. 1
  2. 2
  3. 3
Other examples are having the alphabet like "a" or "A"! (ul) are used to order a collection of items that don't need to be ordered in any specific way. You can change the style of the bullets in unordered lists to squares, dots and circles however the type is defined now with CSS using the list-style-type.
Example for good eating habits:
  • Make sure you wash your hands before eating
  • You should not have your elbows on the table
You can even nest (ul) and (ol) within each other!
Example:
  1. First point
  2. Second point
    • Second point details
    • More details
      1. details inside
      2. details inside
      3. details inside
  3. Third Point
